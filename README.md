# 📱 Sonar App

Sonar App est une application mobile développée avec **React Native + TypeScript**, conçue pour offrir une interface simple et élégante
 avec des cartes interactives et des pages de détails.

---

## Fonctionnalités
✅ Affichage d’une grille de cartes interactives  
✅ Navigation entre la page principale et les pages de détail  
✅ Bouton retour fixe pour une meilleure expérience utilisateur  
✅ Mise en page responsive et design moderne  

---

## 🛠️ Technologies utilisées
- ⚛️ React Native  
- 📘 TypeScript  
- 🎨 StyleSheet 
- 🔧 Git & GitHub pour la gestion de version  

---

## 📦 Installation

Clone le projet puis installe les dépendances :

```bash
git clone https://github.com/klaloi/laloi_ketelove_dev6.git
cd laloi_ketelove_dev6
npm install
````

Lance ensuite l’application :

```bash
npx expo start
```
